Über LCRPD

Der LCRPD Server ist ein Deutschsprachiger Roleplay Server, dass auf dem Roblox Spiel "ERLC" basiert. Der Server befindet sich Gerade im Umbau. Aber wir haben nichts dagegen wenn du mal vorbei schaust. Jeder ist Willkommen!

(Der voll ausgeschriebene Name lautet: Liberty County RP Deutsch)
